# koa的学习

这段时间学习了node，因此特别想实现前后端的结合，由此我选择了koa进行学习。

我的代码实例大部分是来自于[阮一峰的博客](http://www.ruanyifeng.com/blog/2017/08/koa.html)，很感谢他能够提供优秀浅显的教程，另外我还参考了[koa官网](http://koajs.com/)部分实例

## 如何启动

``` nodejs
node ~.js

```
